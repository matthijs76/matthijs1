De 1ste week bij Code Gorilla

Deze week hebben we de geleerde sstof van de zelfstudie maand geoefend. HTML, CSS, JavaScript en PHP gecombineerd in een echte website die we onze portfolio gaan noemen. Het was leerzaam om de geleerde stof te gebruiken. Helaas heb ik de vrijdag gemist omdat ik ziek was. Het ging vrijdag over C.V.'s maken, helaas heb ik weinig om in mijn c.v. te zetten aangezien ik een lange tijd geen reguliere baan heb gehad, wel heb ik de les nog kunnen na lezen in de googledrive. Helaas heb ik de c.v. aan niemand kunnen laten lezen voor een second opinion. Hieronder staat een spelletje wat ik met een tutorial heb gebouwd.ik heb zelf de paddlehit counter en total score toegevoegd. De paddle speed moet nog worden aangepast voor grotere schermresolutie dan mijn laptop(1366*768).
